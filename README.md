# 🌌 From Photons to Patient

An Azure cloud portfolio project that maps physics concepts from 
Stephen Hawking's *A Brief History of Time* to real oncology and 
pharmacy clinical workflows.

## Project Concept
Just as light travels from a star and reveals information about the 
universe, a photon fired in a PET scanner travels through a patient's 
body and reveals a tumor — starting a chain of clinical events that 
mirrors the laws of physics.

## Physics → Clinical Connection Map

| Hawking's Concept | Clinical Application |
|---|---|
| Light / Electromagnetic Spectrum | PET scans, radiation therapy, photodynamic therapy |
| Atoms & Radioactive Decay | Radioactive isotopes in nuclear medicine (Lutetium-177, Fluorine-18) |
| Quantum Mechanics | Chemotherapy disrupting DNA replication at the atomic level |
| The Arrow of Time | Disease staging — cancer progresses in one direction |
| The Big Bang / Origins | Tumor genesis from a single mutated cell |
| Black Holes | Therapeutic event horizon — too much chemo is toxic, too little is ineffective |

## Azure Build Phases

| Phase | Physics Theme | Clinical Story | Azure Service |
|---|---|---|---|
| 1 | Light & Electromagnetic Spectrum | PET scan detects tumor | Resource Group + Blob Storage |
| 2 | Atoms & Radioactive Decay | Treatment plan ordered, pharmacy verifies | Azure Health Data Services (FHIR) |
| 3 | Arrow of Time | Treatment course tracked over time | Azure Monitor + Log Analytics |
| 4 | Observers & Relativity | Different roles see different data | Entra ID + RBAC |
